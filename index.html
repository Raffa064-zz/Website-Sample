Fluxos condicionais em java

O que são fluxos condicionais em java?

Fluxos condicionais são blocos de código especiais que só podem ser executados quando uma determinada condição for satisfeita, dando ao nosso código a capacidade de tomar decisões lógicas a nossa escolha.

Abaixo vemos um exemplo:

1 if (true) {
2 //It will be executed
3 } else {
4 //it will be not executed
5 }
