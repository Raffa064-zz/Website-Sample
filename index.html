Conditional flows in Java

What's conditional flows in Java?

Conditional flows are special blocks of code that can only be executed when a certain condition is satisfied, giving our code the ability to make logical decisions of our choosing.

You can see below an example of it:

1 if (true) {
2 //It will be executed
3 } else {
4 //it will be not executed
5 }
